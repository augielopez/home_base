# Productivity Base

This module is for **personal productivity**: capturing tasks, tracking completion, and keeping a lightweight record of what you intend to do.

## Scope

- **Todos** — Quick capture, mark done, remove; persisted locally until you wire a backend.
- **Future** — Priorities, due dates, projects/tags, or sync (e.g. Supabase) can live under `services`, `stores`, and `types` in this folder.

Use `views`, `components`, `services`, `stores`, and `types` here as the module grows; update this doc when the product scope shifts.

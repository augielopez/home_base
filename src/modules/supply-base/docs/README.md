# Supply Base

This module is for **household groceries and supplies**: what you have on hand, what you need, and when items are no longer good to use.

## Scope

- **Grocery inventory** — Track pantry and fridge items, quantities, locations, and restock signals.
- **Expiration tracking** — Best-by and use-by dates, alerts before spoilage, and safe rotation (first in, first out).

Use `pages`, `components`, `services`, `stores`, and `types` in this folder for implementation; keep this doc updated when the product scope shifts.

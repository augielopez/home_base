# Career Base

This module is for **professional identity and progression**: presenting your experience, targeting roles, and tracking how you grow over time.

## Scope

- **Resume** — Profiles, versions, tailoring, and export or share-ready layouts.
- **Jobs** — Target roles, applications, interviews, and offer tracking.
- **Skills** — Skill inventory, evidence, gaps, and how skills map to roles.
- **Growth** — Learning plans, milestones, certifications, and long-term career direction.

Use `pages`, `components`, `services`, `stores`, and `types` in this folder for implementation; keep this doc updated when the product scope shifts.

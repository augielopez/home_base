# Celebration Base

This module is for **celebrations and gatherings** end to end: planning the event, inviting people, and coordinating what everyone brings.

## Scope

- **Events** — Create and manage celebrations (dates, details, host tools).
- **RSVPs** — Collect responses, headcounts, and attendance status from guests.
- **Guest tiers** — Organize invitees (for example hosts, VIPs, plus-ones, or tiered access to event details).
- **Potluck** — Coordinate shared food and supplies: sign-ups, categories, quantities, and who is bringing what.

Use `pages`, `components`, `services`, `stores`, and `types` in this folder for implementation; keep this doc updated when the product scope shifts.

# Balance Base

This module is for **personal money management**: seeing where money goes, staying on top of obligations, and matching activity to plans.

## Scope

- **Finance** — Accounts, transactions, categories, and the overall financial picture.
- **Bills** — Recurring and one-off bills: amounts, due dates, and payment status.
- **Reconciliation** — Match transactions to bills (and other expected outflows) so the ledger stays accurate.
- **Money flow** — How cash moves between accounts and over time (inflows, outflows, and patterns).

Use `pages`, `components`, `services`, `stores`, and `types` in this folder for implementation; keep this doc updated when the product scope shifts.
